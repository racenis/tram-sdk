Getting started

The source code is available in a software repository at github.com. After downloading the source code, you will need a few libraries before compiling it.

You will need to get the bullet physics library and compile it, then get the glfw library and compile it too, then generate OpenGL 4.0 headers, also get the stb image and glm libraries. Put them all in the libraries/ folder that you get from the repository. That's it for the dependencies.

Then just use CMake. You're supposed to provide your own main.cpp file with the main() function in it, but the engine comes with a basic main() function that has all of the important things in it.
