Getting started

The source code is available in a software repository at github.com. After downloading the source code, you will need a few libraries before compiling it.

You will need to get the bullet physics library and compile it, then get the glfw library and compile it too, as well as the OpenAL-soft, then generate OpenGL 4.0 headers, also get the stb image, stb vorbis and glm libraries. Put them all in the libraries/ folder that you get from the repository. That's it for the dependencies.

Then just use CMake. The CMake script is broken. There should be a project file for the CodeLiter IDE included in the files, but for basic compilation you just need to compile all of the .cpp files in the /src/ directory and sub-directories and link it with -lBulletSoftBody -lBulletDynamics -lBulletCollision -lLinearMath -lglfw3 -lOpenAL32.dll -lglfw3 -lgdi32 -lopengl32 parameters (for gcc at least). You're supposed to provide your own main.cpp file with the main() function in it, but the engine comes with a basic main() function that has all of the important things in it.
