RenderComponent

A RenderComponent renders a single 3D model somewhere in the world.

Usage

Use PoolProxy::New() to allocate a new RenderComponent.

Then use the SetModel(UID("_model_name_here")) to set the model that the RenderComponent will render. Call the UpdateLocation and UpdateRotation methods to set the initial location and rotation of the model.

Finally call the Init() method to start the component. It will load the model in the background and it will start rendering whet that's finished.

When the model is not needed anymore, call the Uninit() method to remove it from the rendering list.

Use PoolProxy::Delete() to deallocate the unneeded RenderComponent.
