Tramway Drifting and Dungeon Exploration Simulator 2022 Software Development Kit Documentation

This is the documentation site for the Tramway Drifting and Dungeon Exploration Simulator 2022 Software Development Kit. If you don't know what that is, then you can find out here.

If you wish to get started using the Tramway Drifting and Dungeon Exploration Simulator 2022 Software Development Kit, then I suggest that you read the engine overview first. After that you can read the getting started guide.

Note

It's not recommended to be using the engine at this time, as it is broken. Most notably, the whole WorldCell level streaming system broke during refactoring, but it should be fixed in a couple of months.
