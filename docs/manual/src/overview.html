Overview

Tramway Drifting and Dungeon Exploration Simulator 2022 Software Development Kit consists of several parts:

  • Engine software library.
  • Model and animation export plugins for the Blender program.

In the future this list will be expanded with:

  • Level editor. 1
  • Graphical configuration editor. 2

Purpose

By using the SDK it's possible to construct tramway drifting and dundeon exploration simulators and other interactive media.

Engine software library

The engine software library is capable of real-time 3D computer graphics, physics simulation via the Bullet physics library, asynchronous asset and level streaming, and is optimised for low-end computers.
