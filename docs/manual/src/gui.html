GUI

GUI code is located in the gui.h file. It has a simple implementation of an immediate mode GUI.

Using

Somewhere in your main render/logic loop you need to put a GUI::Begin() call to start issuing GUI commands for that frame.

After starting the GUI for that rendering frame, you can issue various layout and widget commands. There's GUI::DebugMenu() and GUI::EscapeMenu() commands for premade menus, but you can define your own. All of the widgets are in the gui.h file.

After all of your commands call the GUI::End() to finish the GUI logic and submit the elements for rendering.
