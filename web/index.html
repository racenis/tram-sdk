About

Tramway Drifting and Dungeon Exploration Simulator 2022 Software Development Kit is the engine behind the upcoming computer game Tramway Drifting and Dungeon Exploartion Simulator 2022 where you drift tramways and explore dungeons.

If you'd like to know something more specific, then you can take a look at the manual of the SDK. There's also some screenshots showcasing the capabilities of the SDK.

Features

  • Photo-realistic graphics.
    • Textured 3D models.
    • Gouraud shading.
    • 24-bit color.
    • Night and day conditions.
    • Screen resolutions from 320x200 to 800x600.
    All of that together provides very realistic appearance.
  • Data oriented design.
    • Engine core data is laid out in memory in the most efficient way possible as to increase the data processing speed. The core library also provides templates so that you can have efficient data too.
  • Hyperrealistic physics simulation provided by the Bullet Physics library.
  • LIGMAScript scripting language integration.
  • Open-world style level streaming, including assets.
  • Runs on virtually any computer from the last 15 years.
  • Very fast C++20 code performance.
  • Windows 2009 and Linux support.

How to get

You can download it off of the github.com. You can just download the whole repository. Acually, that's exatcly what you should do.

After that you'll need to get the bullet physics library and compile it, then get the glfw library and compile it too, then generate OpenGL 4.0 headers, also get the stb image and glm libraries. Put them all in the libraries/ folder that you get from the repository. That's it for dependencies. (All this could probably be automated, but I don't have time for that).

Then just use CMake to make the thing. You're supposed to provide your own main.cpp file with the main() function in it, but the engine comes with a basic sample project with some crates.
